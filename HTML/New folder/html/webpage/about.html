Learn About Java Technology
Java is the foundation for virtually every type of networked application and is the global standard for developing and delivering embedded and mobile applications, games, Web-based content, and enterprise software. With more than 9 million developers worldwide, Java enables you to efficiently develop, deploy and use exciting applications and services. From laptops to datacenters, game consoles to scientific supercomputers, cell phones to the Internet, Java is everywhere!
